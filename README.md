# Tooling to help manage user image repos

To install, clone this repo and from inside the directory run:

```
pip install --editable .
```

For the ability to mass-create and merge PRs you will need to install the
[Github cli tool (gh)](https://github.com/cli/cli#installation).

Please see the official [documentation](https://docs.datahub.berkeley.edu/tasks/user-images/managing-multiple-user-image-repos.html)
for detailed instructions.
